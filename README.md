\# ML Classification \& Regression



A collection of supervised machine learning models built in Python using scikit-learn and TensorFlow/Keras. The project is split into two tasks — regression and classification — each comparing three different model types on real-world datasets.



---



\## Task 1 — Regression



\*\*Dataset:\*\* Auto MPG



\*\*Goal:\*\* Predict a vehicle's miles per gallon (MPG) based on engine and weight characteristics.



\*\*Features used:\*\* Weight, Displacement, Horsepower, Cylinders



\*\*Models:\*\*



| Model | File |

|---|---|

| Linear Regression | `regression/linear-regression.py` |

| Random Forest Regressor | `regression/random-forest-regressor.py` |

| Neural Network Regressor | `regression/neural-network-regressor.py` |



\*\*Evaluation Metrics:\*\* Mean Squared Error (MSE), R-squared (R²)



---



\## Task 2 — Classification



\*\*Dataset:\*\* Heart Disease



\*\*Goal:\*\* Predict the likelihood of heart disease based on patient health indicators.



\*\*Models:\*\*



| Model | File |

|---|---|

| Logistic Regression | `classification/logistic-regression-task2.py` |

| Random Forest Classifier | `classification/random-forest-task2.py` |

| Neural Network Classifier | `classification/neural-network-task2.py` |



\*\*Evaluation Metrics:\*\* Accuracy, ROC AUC Score, Confusion Matrix
